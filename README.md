# slop-sloop
AI-generated sailing simulator

What’s in it:

- top-down sailing physics
- wind direction and gusts
- sail trim and rudder controls
- no-go zone / “in irons”
- buoy-rounding score loop
- HUD for speed, heading, trim, heel, VMG, and tack

Controls:

- A / D or ← / →: steer
- W / S or ↑ / ↓: trim sail
- Space: tack assist
- R: restart
